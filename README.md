# Vanesha Hari Portfolio

A dark, recruiter-focused engineering portfolio built with Next.js and ready for Vercel.

## Included
- Responsive one-page portfolio
- Terminal-style hero section
- Experience timeline
- Expandable project cards
- Technical skills cloud
- Leadership and awards
- Downloadable resume
- Scroll reveal animations
- SEO metadata

## Before deploying
Open `app/page.tsx` and replace:
- `https://www.linkedin.com/` with Vanesha's exact LinkedIn URL
- `https://github.com/` with Vanesha's exact GitHub URL

The email is already set to `vaneshahari@gmail.com`.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Deploy to Vercel

1. Upload this folder to a new GitHub repository.
2. Go to Vercel and click **Add New → Project**.
3. Import the GitHub repository.
4. Keep the default Next.js settings.
5. Click **Deploy**.

Vercel will generate a public URL such as:
`vanesha-hari-portfolio.vercel.app`
